Aventure Égyptienne — Version multipage (localStorage)
=========================================================
- Ouvrir index.html dans un navigateur (Chrome/Firefox).
- Bouton "Changer/Entrer mon pseudo" pour saisir un prénom.
- Pages egy1.html ... egy10.html : saisir les mots-codes (ANUBIS, PAPYRUS, SCARABEE, SPHINX, NIL, PHARAON, ANKH, BASTET, PYRAMIDE, OSIRIS).
- Finale : l'image se défloute quand 10/10 sont validés.
- Export Excel .xlsx disponible depuis l'index et la finale.
- Tout est stocké en localStorage (même appareil).

Arborescence :
- index.html
- intro.html
- egy1.html ... egy10.html
- finale.html
- style.css
- app.js

Astuce : pour réinitialiser complètement, utiliser le bouton 'Effacer TOUT' sur l'index.
